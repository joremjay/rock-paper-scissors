# Rock Paper Scissors

A simple Rock Paper Scissors game built with JavaScript as part of The Odin Project Foundations course.

## How it works

The game is played in the browser console.

- The player enters rock, paper, or scissors using a prompt.
- The computer randomly chooses rock, paper, or scissors.
- The game plays 5 rounds.
- The score is shown after each round.
- The final winner is shown at the end.
